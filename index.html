HTML

  • The head element contains information about the webpage which users do not see.
  • The head element is a container for metadata that is used by the browser.
  • The body element represents the visible content shown to the user.
  • Semantic HTML elements describe the role the element plays on the webpage ex "body" and "section".
  • An element refers to the content from the opening tag to closing tag.
  • A tag refers to what is inside the angle brackets ex <head>.

CSS

  • Inline CSS- applies style attributes to a specific HTML tag (not flexible).
  • Internal CSS - contains style elements in the head element of the HTML file. Applies to all tagged elements on the page.
  • External CSS - CSS rules are stored in a separate file. The file is linked to the HTML head element (preferred).
  • Margin - transparent buffer outside an element
  • Padding - transparent buffer inside an element
  • .class is a class selector which is designated by the period.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: adds all files in the working branch
  • git commit -m "commit message describing changes": commit changes to local branch
  • git push origin feature/branch name: syncs local feature branch into GitHub
  • git pull origin main: sync local branch with the remote branch in GitHub
